# JavaScript 高级程序设计第三版读书笔记

# 前言

之前参加过一些面试，根据面试官的建议，感觉自己需要系统的看下 js 基础，为了以后更好地复习和防止遗忘，感觉所以对 js 红宝书中的一些自己感觉容易忽略的细节和一些比较重要的点记录一下，记录的内容只是书中的一小部分，这是一本值得反复咀嚼的好书，希望自己能重视基础。

# 基本概念

  1. <script> 标签中的一些属性
    延迟脚本 defer：相当于告诉浏览器立即下载但是延迟到页面都解析完毕后再执行，用途：表明脚本在执行时不会影响页面的构造； 异步脚本 async：立即下载但是异步执行，同时异步脚本不保证执行的先后顺序；

  2. 严格模式："use strict"

  3. 数据类型
    5 种基本数据类型：Undefined Null Boolean Number String 1 种复杂数据类型：Object

  4. typeof
    typeof null //返回 Object，null 值在逻辑上表示一个空对象指针 typeof undefined //返回 undefined 对未声明和未初始化的变量执行 typeof 会返回 undefined

  5. 各种数据类型转化为 false 的规则
    Undefined undefined Boolean false String "" Number 0 和 NaN Object null

  6. NaN 的特点
    任何涉及 NaN 的操作都会返回 NaN（如 NaN/10） NaN 与任何值都不相等 NaN!=NaN isNaN 函数用来判断一个参数是否可以转换成数值（isNaN("blue")//true 不能转化为数值）

  7. 数值转化
    Number()//用于将任何数据类型转化为数值 '+'在对非数值应用一元加操作符时，该操作符会像 Number()转型函数一样对这个值执行转换 '+'还有拼接字符串的作用，字符串+非字符串时会有类型转化，将非字符串转为字符串 '-'//与'+'规则相同，只是后面转化的数字为负数 parseInt()//用于将字符串转化为数值 parseFloat()//用于将字符串转化为数值

  8. 转化为字符串
    val.toString()//null 和 undefined 值没有这个方法 String(val)//可以将任何类型转化为字符串

  9. 布尔操作符
    同时使用两个逻辑非操作，可以模拟 Boolean()转型函数的行为

  10. 相等和不相等--先转换再比较，全等和不全等--仅比较不转换

  11. for-in 语句
    for-in 语句是一种精准的迭代语句，可以用来枚举对象的属性。因为 ECMAScript 对象的属性没有顺序，所以通过 for-in 循环输出的属性名的顺序是不可预测的。

  12. label 语句
    label 语句用来在代码中添加标签，以便将来使用；还可以和 continue 和 break 连用，用来返回代码中的特定的位置。

  13. with 语句
    with 语句的作用是将代码的作用域设置到一个特定的对象中。严格模式下不允许使用 with，一些大型项目也不建议使用，会导致性能下降。

  14. switch case 语句
    js 中的 switch case 语句和其他语言中的有些不一样，js 中的 switch 语句中可以使用任何数据类型（而其他语言只能使用数值），并且每个 case 语句后面不仅仅可以接常量还可以接变量和表达式。

  15. arguments
    ECMAScript 中函数的参数是一个数组，而且函数体可以通过 arguments 对象来访问这个参数数组，从而获取传递给这个函数的每个参数。arguments.length 可以获取函数参数的个数。
    这个对象还有一个名叫 callee 的属性，该属性是一个指针，指向拥有这个 arguments 对象的函数。

  16. typeof 和 instanceof
    typeof person typeof 是确定一个变量是字符串、数值、布尔值还是 undefined 的最佳工具 person instanceof Object instanceof 用来判断一个值是什么类型的对象（object array regexp）

  17. 内部环境可以通过作用域链访问所有的外部环境，但是外部环境不能访问内部环境中的任何变量和函数

  18. Array 类型常用的方法

    • 检测数组：
      value instanceof Array Array.isArray(value)
    • 转化为字符串：toLocaleString() toString() valueOf()
    • join()可以使用不同的分隔符来构建字符串
    • push()接收任意数量的参数，把他们逐个添加到数组末尾，返回修改后数组的长度
    • pop()从数组末尾移除最后一项，减少数组的 length 值，返回移除的项
    • unshift()在数组前端添加任意个项并返回新数组的长度
    • shift()移除数组中的第一项并返回该项
    • reverse()反转数组项的顺序
    • sort()按升序排列数组项
    • concat()数组项的拼接，对原始数组创建副本，在副本上进行拼接
    • slice()接收一个或两个参数，即要返回数组的起始和结束位置（不包括结束位置[)），若只有一个参数则默认末尾结束
    • splice(起始位置,要删除的项数,要插入的项(可选)) 可以实现删除插入和替换三种功能
    • indexOf(要查找的项,表示查找起点位置的索引（可选）) 返回要查找项在数组中的位置，没有找到返回-1
    • lastIndexOf() 同 indexOf()，查找顺序从右向左
    • 迭代方法：
      every() filtet() forEcah() map() some()

  19. Date()类型
    var now = new Date();

  20. 正则表达式
    var re = /^...$/ re.test(str)

  21. call() apply()和 bind()
    作用：设置函数体内 this 指向 call(this,arguments 对象) apply(this,数组对象) bind()与 call()类似，只是函数定以后不会立即调用需要在函数后再加()才能立即调用

  22. String 类型的常用方法

    • 用于访问字符串中特定字符的方法：charAt()和 charCodeAt() 这两个方法都接收一个参数，即基于 0 的字符位置前者返回给定位置的字符，后者返回给定位置字符的编码。
    • concat()将一个或者多个字符串拼接起来
    • 三个基于子字符串创建新字符串的方法：slice()substr()substring()这些方法都返回一个子字符串，也都接受一个或者两个参数，第一个参数指定子字符串开始的位置，第二个参数指定子字符串最后一个字符后面的位置。
    • 从字符串中查找子字符串的方法：indexOf()和 lastIndexOf()
    • trim()创建一个字符串的副本，删除前置及后缀的所有空格。
    • 字符串的模式匹配方法
      match() 只接受一个参数，要么是一个正则表达式，要么是一个 RegExp()，返回一个数组。 search() 参数要求同 match()，返回字符串中第一个匹配项的索引。 replace(RegExp 或者一个字符串,一个字符串或者一个函数) 如果第一个参数是一个字符串，只替换第一个子字符串，全局替换需要用正则表达式的全局替换/g。
    • split() 可以基于指定的分隔符将一个字符串分割成多个子字符串，并将结果放在一个数组中。

  23. 面向对象

    • 工厂模式
    • 构造函数模式
    • 原型模式
    • 组合使用构造函数模式和原型模式
    • 动态原型模式

  24. 闭包
    闭包指有权访问另一个函数作用域中的变量的函数。创建闭包的常见方式就是在一个函数内部创建另一个函数。

  25. this 对象
    在全局函数中，this 等于 window，而当函数被作为某个对象的方法调用时，this 等于那个对象。 匿名函数的执行环境具有全局性，因此其 this 对象通常指向 window。

  26. 匿名函数

    var function = function(arg0, arg1) {
  //函数体;
};

    (function() {
  //这里是块级作用域；
})(); //定义一个匿名函数并立即调用

# BOM

  1. window 对象

    • setTimeout() 延时一定时间后调用特定代码
    • clearTimeout()
    • setInterval() 按照间隔时间重复执行代码
    • clearInterval()

  2. location 对象 用于获取或者设置窗体的 URL。

  3. navigator 对象 该对象包含很多浏览器的信息，常见的有 userAgent，该属性可以返回由客户机发送服务器的 user-agent 头部的值

  4. history 对象 history 对象与浏览器历史记录进行交互 history.back()后退，history.forward()前进，history.go(1/-1)前进/后退

  5. js 执行机制 先执行栈中的同步任务。 异步任务放入任务队列中。 一旦执行栈中的所有同步任务执行完毕，系统会按次序读取任务队列中的异步任务，于是被读取的异步任务结束等待状态，进入执行栈，开始执行。

# DOM

  1. 获取元素
    • document.getElementById('id)
    • document.getElementsByTagName('标签名') 返回带有指定标签名的对象的集合
    • document.getElementsByClassName('类名') 返回带有指定类名的对象的集合
    • document.querySelector('选择器') 根据指定选择器返回第一个元素对象
    • document.querySelectorAll('选择器') 根据指定选择器返回
  2. 事件基础 事件源（谁） 事件类型（什么事件） 事件处理程序（做啥）
  3. DOM 中的一些操作
  • 创建： document.write innerHTML createElement
  • 增： appendChild insertBefore
  • 删： removeChild
  • 改： 主要修改 dom 元素的内容、属性和表单的值等
  • 查： 主要获取查询 dom 的元素
